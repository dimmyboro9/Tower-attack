Detailed Description

Abstract class that represents all ways to provide the game ( real step game or record )

It has: m_Score - number of attackers who reached the exit m_Map - map on which the game takes place m_Moves - all player's actions m_Routes - paths that attackers walk m_Towers - towers in active game m_Attackers - attackers in active game m_Available_Att - available attackers m_Wait - attackers< who are waiting on enter the game ( not in m_Attackers and not in m_Available_Att )
