Detailed Description

Class that represents the map on which all game actions take place.

It has: m_Length - length of map m_Width - width of map m_Coins - amount of coins on which player can buy attackers m_Num_of towers - the number of towers that are available to the computer m_Models - all map models ( even free ) that together create a whole map m_Entrances - all map entrances and their coordinates m_Exit - coordinates of map exit
