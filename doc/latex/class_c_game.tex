\doxysubsection{Detailed Description}
Abstract class that represents all ways to provide the game ( real step game or record ) 

It has\+: m\+\_\+\+Score -\/ number of attackers who reached the exit m\+\_\+\+Map -\/ map on which the game takes place m\+\_\+\+Moves -\/ all player\textquotesingle{}s actions m\+\_\+\+Routes -\/ paths that attackers walk m\+\_\+\+Towers -\/ towers in active game m\+\_\+\+Attackers -\/ attackers in active game m\+\_\+\+Available\+\_\+\+Att -\/ available attackers m\+\_\+\+Wait -\/ attackers\texorpdfstring{$<$}{<} who are waiting on enter the game ( not in m\+\_\+\+Attackers and not in m\+\_\+\+Available\+\_\+\+Att )
