\doxysubsection{Detailed Description}
Class that represents the map on which all game actions take place. 

It has\+: m\+\_\+\+Length -\/ length of map m\+\_\+\+Width -\/ width of map m\+\_\+\+Coins -\/ amount of coins on which player can buy attackers m\+\_\+\+Num\+\_\+of towers -\/ the number of towers that are available to the computer m\+\_\+\+Models -\/ all map models ( even free ) that together create a whole map m\+\_\+\+Entrances -\/ all map entrances and their coordinates m\+\_\+\+Exit -\/ coordinates of map exit
